Программа «PhotonProcessor» версия 0.1 бета.
Основное предназначение: обработка файлов для 3D-принтера Anycubic Photon, с целью добиться высокой детализации и хорошей прочности изделий одновременно. 
В качестве текущего решения предлагается гибко настраиваемое чередование слоев, с исходными и грубыми границами. Грубые границы в нечетных слоях получаются путем убирания по одному горящему пикселу, в случае если у него не все соседи являются горящими. Точнее количество требуемых соседей настраивается параметром Coarse neighbors. 

В будущем, если производитель выдаст прошивку позволяющую 4-16 оттенков серого для пикселов, вместо радикального Ч/Б, можно будет добавить разные алгоритмы размытия границ, что позволит вывести качество печати на ещё более высокий уровень.

Программа распространяется как есть, с исходным кодом свободным для изменений и коммерческой эксплуатации. Для взаимодействия с файлами принтера, был портирован код проекта https://github.com/Photonsters/PhotonFileEditor
